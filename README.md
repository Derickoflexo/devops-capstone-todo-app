# DevOps Capstone Project (GitHub Actions Edition)

This project deploys a Todo Application on AWS using:

- Terraform (Infrastructure Provisioning)
- Ansible (Server Configuration & Deployment)
- Docker (Containerization)
- GitHub Actions (CI/CD Pipeline)

Folder Structure:
- app/frontend  → React Todo UI
- app/backend   → Node Todo API
- infra/terraform → AWS Infrastructure
- infra/ansible → Config & Deploy
- .github/workflows/deploy.yml → CI/CD Pipeline
